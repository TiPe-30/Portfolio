Les langages C./C++ sont des langages utilisés pour la programmation sur les systèmes d'exploitation d'ordinateur! Leurs principaux avantages étant la performance, car ce sont les langages parmi les plus rapides qui existent. Pas mal de petit projet que j'ai entrepris notamment le codage d'un petit jeu vidéo, m'a permis de monter en compétences sur ces domaines. J'ai pour ambition d'utiliser ces langages dans certains de mes projets personnels pour pouvoir augmenter les performances de mes applications.

HTML : HyperTextMarkupLanguage est un langage de balisage, qui permet de décrire les élements de la page(Vidéos, image, texte) mais pas de les mettre en forme. Pour cela il existe un autre language nommé CSS pour CascadingStyleSheet qui sont des feuilles de style permettant de mettre en forme le contenu de la page :

  • Position des élements
  • Couleur
  • animation
  • police
  • etc..

Et pour finir, il y a un autre language qui cet fois lui de programmation : Le Javascript. Et qui lui permet de programmer des veritables logiciels a l'intérieur des pages web, ou encore faire des traitements ou inclure des fonctionnalités beaucoup plus avancés. Les logiciels web on pour principal avantages la portabilité, tous les sytème modernes sont dotées de navigateur web tel que les smartphones ou les ordinateurs, ect.. L'application alors créer, est facile a déployer. Cependant, une connexion internet est requise, on ne peut pas tout avoir!
