Projet

Des projets sont créés par mon établissement, afin d'avoir une première approche du monde de l'entreprise. Ces projets sont nommés SAÉ(Situation Apprentissage Evalutation) et une d'entre elle m'a particulièrement interessé! L’objectif de cet SAÉ était de créer un programme de classification automatique des dépêches selon la plus forte probabilité qu’une dépêche appartiennent a une catégorie. 5 catégories de dépêches étaient présentes :

  • SPORT
  • CULTURE
  • SCIENTIFIQUE
  • ÉCONOMIE
  • POLITIQUE

Pour chacune des dépêches donnée le programme devait donner un taux d’appartenance d’une dépêche a une des ces catégories, c’était donc un programme d’apprentissage automatique, qui permettait a partir d’un fichier texte d’apprentissage ou des dépêches sont déjà présentes, de calculer le taux de chaque dépêche d’un autre fichier donné par un utilisateur par exemple pour avoir le taux d’appartenance d’une dépêche. Ce projet a vu le jour dans mon IUT en Janvier 2024, Nous avons utilisé le langage JAVA pour implémenter ce besoins. Besoins, qui mobilisait notamment les compétences suivantes :
Compétence 1(Implémenter un besoins clients), 2(Optimiser des applications informatiques) ou nous avons utilisés différents procédés d’optimisation pour accélérer le traitement des dépêches. Ce travail était a réaliser en groupe de 2, je m’occupais principalement de la partie code. Une autre partie consistait a réaliser un rapport en anglais concernant la SAE. Nous avons été au-dela de l’attendu en intégrant des techniques d’apprentissage automatique supplémentaire visant a enrichir le programme.

Sur les plateformes windows des logiciels existent déjà afin de nettoyer l'ordinateur de fichier inutile. Vous les connaissez sans doute déjà : CCleaner, Malwerbytes, etc.. Sous linux il existe déjà des outils intéressant tel que : BleachBit qui permet de supprimer des fichiers temporaires ou autres. Le but étant de créer un logiciel ressemblant, plus aboutis, avec en prime des analyse plus poussée grâce a un programme d'apprentissage automatique qui repère les fichiers inutiles sur le disque, et les fichiers corrompus. Automatiser cet outil serait un grand plus, ainsi que proposer une interface graphique conviviale dans un futur. Ce projet serait réalisé en java et en script bash, puis une version C/C++ vera le jour pour augmenter les possibilitées avec l'application. Certaine partie en assembleur pourrait être codée afin de rendre l'application la plus rapide que possible!
